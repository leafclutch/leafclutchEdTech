REQUIREMENTS

What You Need to Get Started

New to data science? No worries — this program is designed for absolute beginners.

Beginner Friendly

  • No prior programming or statistics experience needed
  • Basic computer and spreadsheet skills
  • Curiosity about data and patterns
  • High school math is helpful but not required

Technical Requirements

  • A laptop with at least 4GB RAM
  • Stable internet connection
  • Python & Jupyter Notebook (installed in first session)
  • Google Colab (free) as a cloud alternative
CURRICULUM

Course Content

Master Data Analysis with Python, Statistics, SQL, Power BI, Tableau, EDA, Feature Engineering & real-world projects.

8 sections • 24 lectures • 3 months total length
  • Python basics, data types & control flow
  • Pandas, NumPy & essential libraries
  • Data manipulation & analysis with Python
  • Descriptive statistics & distributions
  • Probability distribution functions
  • Inferential stats & hypothesis testing
  • Handling missing data & encoding
  • Feature scaling & selection techniques
  • Data preprocessing & transformation
  • Data visualization with Matplotlib & Seaborn
  • Identifying trends, patterns & outliers
  • Storytelling with data visualizations
  • SQL fundamentals to advanced queries
  • SQL Server, JOINs & aggregation
  • Working with large datasets efficiently
  • Interactive dashboards & DAX calculations
  • Data transformations & reporting
  • Tableau visualization & business insights
  • Advanced Excel for data analysis
  • ETL processes & data workflows
  • AWS & Snowflake cloud analytics
  • End-to-end data analysis projects
  • Real-world dataset analysis & reporting
  • Portfolio building & career preparation
