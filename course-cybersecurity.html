REQUIREMENTS

What You Need to Get Started

Interested in cybersecurity but not sure where to start? This program is designed for beginners.

Beginner Friendly

  • No prior cybersecurity or hacking experience needed
  • Basic understanding of how computers and the internet work
  • Curiosity about how systems can be secured and exploited
  • Basic command-line familiarity is helpful but we teach it too

Technical Requirements

  • A laptop with at least 8GB RAM (for running virtual machines)
  • Stable internet connection
  • VirtualBox (free) for running Kali Linux in a sandbox
  • All tools (Wireshark, Nmap, Burp Suite) installed during class
CURRICULUM

Course Content

Master Ethical Hacking from scratch — Kali Linux, Nmap, Metasploit, Web Pentesting, WiFi Cracking & real-world exploits.

8 sections • 24 lectures • 3 months total length
  • Kali Linux, VirtualBox & VM setup
  • Linux terminal & system navigation
  • Networking fundamentals & protocols
  • Python 3 essentials for security
  • Building a custom port scanner
  • Automating security tasks with scripts
  • Passive & active information gathering
  • Whois, theHarvester & Sherlock tools
  • Email scraping & OSINT techniques
  • TCP/UDP scanning & Nmap mastery
  • OS detection & service enumeration
  • Firewall evasion & stealth scans
  • Nessus vulnerability scanner setup
  • Searchsploit & Nmap scripts
  • CVE research & exploit databases
  • Metasploit Framework & payloads
  • Reverse shells & bind shells
  • Windows & Linux exploit techniques
  • SQL injection & XSS attacks
  • Command injection & CSRF exploits
  • Burpsuite & web app pentesting
  • Persistence, privilege escalation & Meterpreter
  • WiFi cracking, MITM & social engineering
  • Report writing & defensive strategies
